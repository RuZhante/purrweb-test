## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

Purrweb test (RESTful API с авторизацией для Trello на Nest.js)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## TypeORM

```bash
# create migration
$ npm run db:create #add name

# run migration
$ npm run db:migrate

# drop migrations
$ npm run db:drop
```
